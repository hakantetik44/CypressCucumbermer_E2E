## 🐛 Dépannage

### Problèmes Courants

1. **Erreur de Timeout**
   ```bash
   # Augmenter le timeout dans cypress.config.js
   defaultCommandTimeout: 10000
   ```

2. **Échec de Screenshots**
   ```bash
   # Vérifier les permissions du dossier
   chmod -R 777 cypress/screenshots
   ```

3. **Problèmes de Vidéo**
   ```bash
   # Nettoyer le cache
   npm run cy:clean
   ```
